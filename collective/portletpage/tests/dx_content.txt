================
Creating content
================

The portlet pages are simple Dexterity content. Let's ensure that they can
be created, using a test-browser test.

First, some set-up.

    >>> from plone.testing import z2
    >>> browser = z2.Browser(layer['app'])
    >>> portal = layer['portal']
    >>> portal_url = portal.absolute_url()
    
    >>> browser.handleErrors = False
    >>> portal.error_log._ignored_exceptions = ()
    
We will log authorise as the portal owner.
    
    >>> from plone.app.testing import SITE_OWNER_NAME
    >>> from plone.app.testing import SITE_OWNER_PASSWORD
    >>> browser.addHeader('Authorization',
    ...                   'Basic %s:%s' % (SITE_OWNER_NAME, SITE_OWNER_PASSWORD))
    
Start on the portal root.
    
    >>> browser.open(portal_url)
    
Now add a portlet page

    >>> browser.getLink(id='dxportlet-page').click()
    
Fill in the form and submit.
    
    >>> browser.getControl(name='form.widgets.IDublinCore.title').value = "Welcome"
    >>> browser.getControl(name='form.widgets.IDublinCore.description').value = "This is the front page"
    >>> browser.getControl(name='form.widgets.IRichTextBehavior.text').value = "<b>Hello</b>"
    >>> browser.getControl('Save').click()
